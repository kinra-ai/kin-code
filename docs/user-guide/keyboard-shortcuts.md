# Keyboard Shortcuts

Quick reference for all keyboard shortcuts in Kin Code.

## Input Shortcuts

| Shortcut | Action |
|----------|--------|
| `Enter` | Submit message |
| `Ctrl+J` | Insert newline |
| `Shift+Enter` | Insert newline (terminal-dependent) |

## Navigation

| Shortcut | Action |
|----------|--------|
| `Escape` | Interrupt agent or close dialogs |
| `Ctrl+C` | Quit (or clear input if text present) |

## View Controls

| Shortcut | Action |
|----------|--------|
| `Ctrl+O` | Toggle tool output view |
| `Ctrl+T` | Toggle todo list view |

## Mode Toggle

| Shortcut | Action |
|----------|--------|
| `Shift+Tab` | Cycle through modes (Default → Plan → Accept Edits → Auto Approve) |

## Tool Approval

When prompted for tool approval:

| Key | Action |
|-----|--------|
| `y` | Yes - approve this execution |
| `n` | No - deny this execution |
| `a` | Always - always approve this tool |
| `v` | Never - never allow this tool |

## Terminal Configuration

If `Shift+Enter` doesn't work for newlines, run `/terminal-setup` to configure your terminal emulator.

## Special Input Features

| Feature | Usage |
|---------|-------|
| File autocomplete | Type `@` followed by a path |
| Direct shell command | Prefix with `!` (e.g., `!git status`) |
| Slash commands | Start with `/` (e.g., `/help`) |
