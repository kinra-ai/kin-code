# Installation

Kin Code requires Python 3.12 or higher.

## Using uv (Recommended)

[uv](https://docs.astral.sh/uv/) is a fast Python package manager. Install Kin Code as a tool:

```bash
uv tool install kin-code
```

## Using pip

```bash
pip install kin-code
```

## Verifying Installation

After installation, verify Kin Code is available:

```bash
kin --version
```

## Next Steps

Once installed, proceed to [Quick Start](./quickstart.md) to configure your API key and start using Kin Code.
