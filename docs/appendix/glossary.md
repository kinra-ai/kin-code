# Glossary

## Key Terms and Definitions

### Agent
A configured instance of Claude with specific prompts, capabilities, and behaviors for different workflows.

### ACP (Agent Client Protocol)
A protocol for integrating Kin Code with code editors like Zed, JetBrains, and Neovim.

### MCP (Model Context Protocol)
A protocol for connecting external tools and services to Kin Code.

### Skill
Reusable extensions to Kin Code that provide custom commands and functionality.

### Subagent
A delegated agent spawned to handle a specific task independently.

### Trust Folder
A folder explicitly marked as trusted, allowing Kin Code to perform operations without confirmation.

### Tool
A built-in capability of Kin Code for file operations, command execution, search, etc.

### Session
A conversation history with the AI, including context and settings.

### Slash Command
A command prefixed with `/` that provides special functionality in interactive mode.
