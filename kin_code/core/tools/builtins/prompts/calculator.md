Use `calculator` to evaluate mathematical expressions accurately.

**Supported operations:**
- Arithmetic: `+`, `-`, `*`, `/`, `%`, `**` (power)
- Functions: `sin`, `cos`, `tan`, `sqrt`, `log`, `log10`, `log2`, `exp`, `abs`, `pow`, `floor`, `ceil`, `round`, `min`, `max`
- Constants: `pi`, `e`, `tau`
- Parentheses for grouping

**Examples:**
- `calculator(expression="2 + 3 * 4")` -> 14
- `calculator(expression="sqrt(16) + sin(pi/2)")` -> 5.0
- `calculator(expression="log(e**2)")` -> 2.0
- `calculator(expression="max(10, 20) + min(5, 3)")` -> 23
